== Description ==

Accept Credit card, Debit card and Bank account payment directly on your store with the official Flutterwave for WooCommerce.

Flutterwave is available in:

* __Nigeria__
* __Ghana__
* __Kenya__
* __Uganda__
* __Tanzania__
* __Rwanda__
* __South Africa__
* __Zambia__
* __Sierra Leone__



== Installation ==

== Requirements ==
1. WooCommerce

== Manual Installation ==
1. Upload `Flutterwave-Woocommerce.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Automatic Installation ==
1. Click on "Plugins > Add New" from the left menu.
2. Search for "Flutterwave" and click on Install.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to the Flutterwave settings page and enter your API keys.
